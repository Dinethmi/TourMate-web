# 🌍 TourMate – Travel Management Application

**TourMate** is a web-based travel management application developed to simplify the travel planning experience. The system enables users to browse available tour packages, view destination details, make bookings, and manage their travel information through an intuitive and user-friendly interface.
The application was developed as an academic group project following software development best practices, providing practical experience in full-stack web development, database management, and collaborative teamwork.
